Schedule Predictions
  

Upload Prediction Data

To make predictions, you first need to upload a prediction dataset:

  • Navigate to the Upload File page.
  • Ensure that your file is in CSV format and under the 10MB limit.
  • Follow the steps to upload the data.

 

Create Prediction Dataset

After the file is uploaded, proceed to the Dataset Page to create the prediction dataset from the uploaded data.

Map Training Model to Prediction Dataset

Finally, link your trained model to the prediction dataset:

  • Go to the Model Schedule Page and select your trained model.
  • Assign it to the prediction dataset to begin generating predictions.
